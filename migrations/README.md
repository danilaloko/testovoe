# Миграции базы данных

Эта папка содержит SQL-скрипты миграций для управления схемой базы данных.

## Формат именования файлов

Миграции должны называться в формате: `XXX_description.sql`

Где:
- `XXX` - порядковый номер миграции (001, 002, 003, ...)
- `description` - краткое описание миграции на английском или транслите

Примеры:
- `001_create_grades_table.sql`
- `002_add_email_column.sql`
- `003_add_index_on_subject.sql`

## Как создать новую миграцию

1. Создайте новый файл в папке `migrations/` с номером следующей миграции
2. Примените миграцию командой: `python migrate.py`

## Откат миграций

Текущая система не поддерживает автоматический откат. Для отката нужно:
1. Создать новую миграцию с обратными изменениями
2. Или вручную удалить запись из таблицы `schema_migrations` и выполнить SQL для отката

## Проверка статуса миграций

```bash
python migrate.py
```

Скрипт покажет:
- Сколько миграций применено
- Какие миграции ожидают применения
- Результат применения новых миграций
